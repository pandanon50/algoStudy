### 알고리즘 공부 방법

1. 알고리즘 관련 지식(이론) 공부해 문서로 남기기
2. 문제를 풀고 양식에 맞추어 저장
3. 주말을 이용해 피드백을 요청하고 (혹은 이슈 남기기)
4. 피드백을 이해 하고 다시 풀기
5. 피드백할 때 배운 개념 공부해 문서로 남기기
